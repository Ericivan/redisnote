#### Redis String 类型基本操作

1. 设置value值


   ```powershell
   set age 10
   ```

   ​

2. 获取value值

   ```shell
   get age
   ```

   ​

3. 自增操作

   ```powershell
   incr age
   ```

   ​

4. 自减操作

   ```Shell
   decr age
   ```

   ​

5. 自增自定义数值操作

   ```shell
   incrby age 10
   ```

   ​

6. 自增自定义数值操作

   ```shell
   decrby age 10
   ```

   ​
